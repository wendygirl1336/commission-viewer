# 한국보험금융 보험 수수료 조회 시스템

엑셀 파일로 관리되는 보험 상품별 수수료 데이터를 웹에서 조회할 수 있도록 만든 Python 기반 웹 애플리케이션입니다.  
관리자는 엑셀 파일을 업로드하고, 조회자는 로그인 후 회사별·상품별 수수료 정보를 검색/필터링해서 확인할 수 있습니다.

![한국보험금융 로고](assets/kif-logo.png)

---

## 프로젝트 개요

이 프로젝트는 한국보험금융 내부 수수료표 조회를 위한 웹 서비스입니다.

업로드된 엑셀 파일에서 다음 항목을 자동 추출합니다.

- 회사명 / 보험사명
- 상품명
- 1차년 수수료
- 2차년 수수료
- 3차년 수수료
- 총수수료
- 원본 시트 구분

조회 화면에서는 회사명 또는 상품명으로 검색할 수 있고, 회사별/구분별 필터링도 가능합니다.

---

## 주요 기능

### 로그인 기능

- 관리자 계정 로그인
- 조회 계정 로그인
- 세션 기반 로그인 유지
- 로그인 실패 5회 이상 시 일정 시간 잠금
- CSRF 토큰 검증 적용

### 엑셀 업로드

- 관리자만 엑셀 업로드 가능
- `.xlsx`, `.xlsm`, `.xltx`, `.xltm` 형식 지원
- 구형 `.xls` 파일은 지원하지 않음
- 업로드한 데이터는 `current_upload.json`에 저장됨

### 수수료 조회

- 회사명 또는 상품명 검색
- 회사별 필터
- 원본 시트/구분 필터
- 표시 행 수, 회사 수, 상품 수, 총수수료 요약 표시
- 조회 계정은 데이터 수정 불가

### 관리자 기능

- 엑셀 업로드
- 업로드 데이터 전체 삭제
- 관리자 화면에서 행 수정/삭제 UI 제공

---

## 기술 스택

- Python 3.12.13
- Python 기본 `http.server`
- openpyxl 3.1.5
- HTML
- CSS
- JavaScript
- Render 배포 설정 지원
- Procfile 기반 배포 지원

별도의 Flask, Django, React 없이 단일 Python 서버와 정적 HTML로 구성되어 있습니다.

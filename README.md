# 보험 수수료 조회·계산 시스템

엑셀로 관리되는 보험 상품별 수수료율을 웹에서 조회하고, 가입 금액을 입력해 연차별 예상 수수료를 즉시 계산할 수 있는 단일 파일 기반 웹 애플리케이션입니다.

관리자는 엑셀 파일을 업로드해 데이터를 갱신하고, 조회 사용자는 회사명·상품명·구분 필터로 필요한 수수료 정보를 빠르게 확인할 수 있습니다. 엑셀에서 퍼센트로 표시된 값은 `1.383` 같은 내부 숫자로 노출하지 않고 `138.3%`처럼 원래 표시 단위를 유지합니다.

## 핵심 기능

- `.xlsx`, `.xlsm`, `.xltx`, `.xltm` 형식의 엑셀 수수료표 업로드
- 생명보험/손해보험/월납 매트릭스 형태의 표 자동 파싱
- 회사명, 상품명, 원본 시트 구분 기준 검색 및 필터링
- 엑셀 퍼센트 셀의 표시 단위 보존
- 표 행 클릭 시 수수료 계산 팝업 제공
- `10만원`, `1,000,000`, `1억 2천만원` 같은 금액 입력 자동 변환
- 1차년부터 4차년까지 연차별 수수료율과 계산 금액 표시
- 관리자 전용 업로드, 수정, 삭제 기능
- 조회 계정용 읽기 및 계산 기능

## 계산 UX

수수료율 행을 클릭하면 배경이 흐려진 팝업이 열립니다. 금액을 입력하면 다음 항목이 한 화면에 정리됩니다.

- 회사
- 상품
- 입력 금액
- 1차년 수수료율 및 계산 금액
- 2차년 수수료율 및 계산 금액
- 3차년 수수료율 및 계산 금액
- 4차년 수수료율 및 계산 금액
- 총 수수료율 및 총 계산 금액
- 원본 구분

예를 들어 입력 금액이 `10만원`이고 1차년 수수료율이 `138.3%`이면 1차년 계산 금액은 `138,300원`으로 계산됩니다.

## 보안 설계

- 세션 기반 로그인
- 관리자 업로드/수정/삭제 권한 분리
- CSRF 토큰 검증
- 로그인 실패 횟수 제한 및 임시 잠금
- 업로드 파일 크기 제한
- JSON 요청 본문 크기 제한
- 주요 보안 헤더 적용
- 정적 파일 경로 탐색 차단

보안을 위해 계정 정보는 문서에 공개하지 않습니다.

## 기술 스택

- Python 3.12
- Python 표준 `http.server`
- openpyxl
- HTML/CSS/JavaScript
- Render 배포 설정

별도 프레임워크 없이 작은 서버와 정적 UI로 구성해 배포와 유지보수가 단순합니다.

## 프로젝트 구조

```text
.
├── app.py              # 서버, 인증, 엑셀 파싱, 업로드 API
├── index.html          # 조회 화면, 필터, 계산 팝업 UI
├── requirements.txt    # Python 의존성
├── runtime.txt         # 런타임 버전
├── render.yaml         # Render 배포 설정
├── Procfile            # 프로세스 실행 설정
├── package.json        # npm start 연결
└── assets/
    └── kif-logo.png    # 서비스 로고 이미지
```

## 실행

```bash
pip install -r requirements.txt
python app.py
```

실행 후 브라우저에서 `http://127.0.0.1:8766`으로 접속합니다.
